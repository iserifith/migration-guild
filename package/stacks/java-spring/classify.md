# Legacy Framework Classifier

Inspect imports, annotations, XML descriptors, packaging, and build signals. Classify each artifact as a REST endpoint, servlet/filter, service bean, exception handler, startup/config, utility, model, or test. Record the detected framework, destination type, complexity, configuration dependencies, and tests-first strategy.

Signals include JAX-RS, servlets, EJB, Struts, legacy Spring MVC, `web.xml`, `applicationContext.xml`, `struts.xml`, and `ejb-jar.xml`.
